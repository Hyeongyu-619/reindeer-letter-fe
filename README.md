# 오늘의 기억을 선물하는 편지, 순록의 편지 💌

과거의 친구, 가족, 나 자신에게 메시지를 받을 수 있는 타임캡슐 앱 <b>"순록의 편지"</b> 입니다.
순록의 편지로 추억을 현재로 불러오는 순간의 설렘을 느껴보세요.

## 주요 기능

#### 📌 특정 날짜를 선택해 미래의 자신, 친구 혹은 가족에게 편지를 작성할 수 있습니다.
